# 禁言 (minutes) (@sb.) \[minutes] (@sb.) ...

看着好像有点复杂？我举个例子：

```text
禁言 1 @114514 2 @1919810 3 @homo
```

那么就会发生：

| 谁？      | 被禁言了多少分钟？ |
|---------|-----------|
| 114514  | 1         |
| 1919810 | 2         |
| homo    | 3         |

# 踢出 (@sb.) \[@sb.] ...

```text
踢出 @114514 @1919810 @homo
```

那么 114514 1919810 homo 就会被踢出
